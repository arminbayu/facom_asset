Preparation

Before accessing FasaPay Mobile Application, please make sure that you are registered as a FasaPay member with at least an Active status (Registered status cannot use FasaPay Mobile Application).

Get an Auth Key :
1. Log in to FasaPay member area: https://fasapay.com/ Login.
2. Select Account Access (API) > Application API.
3. Click Authorize New App to add a new device.
4. Fill in your Application name, then click CREATE.
5. Auth Key successfully created. Save the AUTH_KEY, you will need it later to log in to FasaPay Mobile Application.

Installation

1. Install FasaPay Mobile Application.
2. Open FasaPay Mobile Application. You can click NEXT or SKIP the slider.
3. Scan the QR code or input the AUTH_KEY manually.
4. Create a PIN and input once again in the CONFIRM PIN field, then click SUBMIT. Please make sure to remember/save your PIN.
5. FasaPay Mobile Application is ready to use.
