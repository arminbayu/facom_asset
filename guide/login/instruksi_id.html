Persiapan

Sebelum mengakses Aplikasi FasaPay Mobile, pastikan bahwa Anda telah terdaftar sebagai anggota FasaPay dengan status Aktif (status Terdaftar tidak dapat menggunakan Aplikasi FasaPay Mobile).

Dapatkan Kunci Auth :
1. Masuk ke member area FasaPay: https://fasapay.com/ Masuk.
2. Pilih Akses Akun (API)> Application API.
3. Klik Authorize New App untuk menambahkan perangkat baru.
4. Isi nama Aplikasi Anda, lalu klik BUAT.
5. Auth Key berhasil dibuat. Simpan AUTH_KEY, Anda akan membutuhkannya nanti untuk masuk ke Aplikasi FasaPay Mobile.

Instalasi

1. Instal Aplikasi FasaPay Mobile.
2. Buka Aplikasi FasaPay Mobile. Anda dapat mengklik NEXT atau SKIP slider.
3. Pindai kode QR atau masukkan AUTH_KEY secara manual.
4. Buat PIN dan masukan sekali lagi di kolom KONFIRMASI PIN, lalu klik KIRIM. Harap pastikan untuk mengingat / menyimpan PIN Anda.
5. Aplikasi FasaPay Mobile siap digunakan.
