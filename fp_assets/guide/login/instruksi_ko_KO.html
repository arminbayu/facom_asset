준비

FasaPay 모바일 애플리케이션에 액세스하기 전에 다음을 확인하십시오. 최소한 FasaPay 회원으로 등록되어 있습니다. 활성 상태 (등록 상태는 FasaPay를 사용할 수 없습니다. 모바일 애플리케이션).

인증 키 가져오기:
1. FasaPay 회원 영역에 로그인: https://fasapay.com/ 로그인.
2. 계정 액세스(API) > 애플리케이션 API 선택.
3. 새 앱 승인을 클릭하여 새 장치를 추가합니다.
4. 애플리케이션 이름을 입력한 다음 CREATE를 클릭합니다.
5. 인증 키가 생성되었습니다. AUTH_KEY를 저장하십시오. 필요합니다. 나중에 FasaPay 모바일 애플리케이션에 로그인합니다.

설치

1. FasaPay 모바일 애플리케이션을 설치합니다.
2. FasaPay 모바일 애플리케이션을 엽니다. 다음 또는 건너뛰기를 클릭할 수 있습니다. 슬라이더.
3. QR 코드를 스캔하거나 인증 키를 수동으로 입력하십시오.
4. 핀을 생성하고 핀 확인 필드에 다시 한 번 입력하고, 그런 다음 제출을 클릭하십시오. 반드시 기억/저장하십시오. 핀.
5. FasaPay 모바일 애플리케이션을 사용할 준비가 되었습니다.
