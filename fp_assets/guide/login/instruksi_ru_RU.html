Подготовка

Перед входом в мобильное приложение FasaPay убедитесь, что вы зарегистрированы в качестве члена FasaPay по крайней мере Статус Активный (статус Зарегистрированный не может использовать FasaPay Мобильное приложение).

Получите ключ авторизации:
1. Войдите в личный кабинет FasaPay:https://fasapay.com/ Авторизоваться.
2. Выберите Доступ к учетной записи (API) > API приложения.
3. Нажмите Авторизовать новое приложение, чтобы добавить новое устройство.
4. Введите имя приложения, затем нажмите СОЗДАТЬ.
5. Ключ авторизации успешно создан. Сохраните ключ авторизации, он вам понадобится позже, чтобы войти в мобильное приложение FasaPay.

Монтаж

1. Установите мобильное приложение FasaPay.
2. Откройте мобильное приложение FasaPay. Вы можете нажать «ДАЛЕЕ» или «ПРОПУСТИТЬ». ползунок.
3. Отсканируйте QR-код или введите ключ авторизации вручную.
4. Создайте пин-код и введите его еще раз в поле Подтвердить пин-код, затем нажмите ОТПРАВИТЬ. Пожалуйста, не забудьте запомнить/сохранить ШТЫРЬ.
5. Мобильное приложение FasaPay готово к использованию.
